HTML/CSS Cheatsheet

HTML

Text Tags

Tag Name Description
<h1-h6> Headings 1-6 h1 is the largest, h6 is the smallest
<p> Paragraph text body
<b> Bold Increases the font weight
<i> Italic Changes font style
<sup> Superscript Puts characters in top right corner like an exponent
<sub> Subscript Puts characters in bottom right corner like chemical formulas
<br> Line break Following starts on a new line(no closing tag)
<hr> Horizontal rule A horizontal line separating two sections(no closing tag)
<strong> Strong Semantic markup for bold
<em> Emphasis Semantic markup for italic
<blockquote> Long quote(entire paragraph) Indents text on both sides
<q> Quote Adds quotation marks around text
<abbr> Abbreviation/Acronyms Use "title=" for full term, which will display when hovered over
<cite> Citation Content is rendered in italics
<dfn> Definiton Content is rendered in italics
<address> Address Contains contact details nested within
<ins> Insert Underlines inserted content
<del> Delete Strikes a line through the center of the deleted content
<s> Strike Strikes a line through the center of the content (ie. old price, still want it visible)

List Tags

Tag Name Description
<ol> Ordered List Box in which list items are nested and numbered in decending order
<ul> Unordered List Box in which list items are nested and marked with bullet points
<li> List Items Content is indented and either numbered or marked with bullet points
<dl> Definition List Box in which the term and the defintion are nested
<dt> Definition Term The term being defined, set apart from the following defintion
<dd> Definition The definition of the previous term, indented and a line below from the term

Anchor Tags

Tag Name Description
<a> Anchor(link) Provides a link using (href=attribute/url/relative path)

Anchor Attributes

Attribute Content Description
mailto: href="mailto:email@address.com" Link opens user's email with the designated destination address
target href="www.example.com" target="_blank" opens link in a new window

Table Tags

Tag Name Description
<table> Table Box in which table elements are nested
<thead> Table Header Box in which table headings are nested
<tbody> Table Body Box in which the main body of the table is nested
<tfoot> Table Footer Box in which the the table results data are nested
<tr> Table Row Box in which the table heading and table data is nested
<th> Table Heading Element in which the heading data is entered(displayed in bold)
<td> Table Data Element in which the table data is entered

Table Attributes

Attributes Content Description
colspan colspan="number of columns to span" Extends how far the data spans horizontally in the table
rowspan rowspan="number or rows to span" Extends how far the data spans vertically in the table

Form Tags

Tag Name Description
<form> Form Box that contains elements to create/control the filling out of a form
<input> Input Element used to create several different form controls
<textarea> Text Area Element used to create a multi-line text input
<select> Select Creates a drop down list box
<option> Option Element used to specify the options that a user can select from
<button> Button Creates a button that you can combine with other elements(ie. <img>)
<label> Label Element to be used in conjunction with all other form controls, making them more accessible to vision-impaired users
<fieldset> Fieldset Element used to group a set of controls together, and provides a border around them
<legend> Legend Element nested inside the fieldset, used to name the group of controls inside

Form Attributes

Attributes Content Description
action action="www.example.com" The URL for the site that will receive the information filled out in the form
method method="get" Values from the form are added to the end of the URL. Ideal for short forms, or when you're only retrieving data from the web server
method method="post" Values are sent in what are known as HTTP headers. Use for long forms, contains sensitive data, allows users to upload files, or if it adds/deletes information from a database
type type="text" Creates a single line text input
type type="password" Just like type="text" except the characters are blocked out
type type="radio" Allows users to choose just one of multiple options
type type="checkbox" Allows user to select/unselect one or more options in answer to a question
type type="file" Creates a box with an upload button, allowing the user to select a file from their device
type type="submit" Used to send a form to the server
type type="image" Allows user to use an image for their submit button
type type="hidden" Allows author to add values to forms that the users cannot see
type type="date" Allows user to select a date to submit on the form
type type="email" Only allows valid email address formats to be submitted
type type="url" Only allows valid URL format to be submitted
type type="search" Creates a search box
name name="name of information being sent" Tells the server what information is being submitted(ie. name="username")
maxlength maxlength="example number" Limits the number of characters that a user can submit
value value="name of checkbox/radio etc" Tells the server what information the user has submitted
checked checked="example box" Indicates which value should be selected when the page loads
selected selected="selected" Similar to "checked" attribute, but for drop down lists
size size="example number" Creates a box for the drop down list, the number tells how many options the box should show at a time
multiple multiple="multiple" Allows user to select multiple options
for for="form control name" It states which form control the label belongs to
placeholder placeholder="example text" Displays the text in the text box upon loading the page

Extra Markup Tags

Tag Name Description
<!DOCTYPE html> Doctype Declares the language being used in the following code
<!-- --> Comment Comments made in the code for future reference, anything inside the comment tag will not be processed
<div> Division Allows you to group a set of elements together in one block-level box
<span> Span It is the inline equivalent of the <div> element
<iframe> Inline Frame It's a little window that's been cut into your page, and in that window, you can see another page
<meta> Meta Data Lives inside the <head> element, and contains information about that web page

Extra Markup Attributes

Attributes Content Description
id id="example" Every HTML element can carry the id attribute. It is used to uniquely identify that element from other elements on the page
class class="example" Every HTML element can carry a class attribute. Unlike id, class is used to identify several elements as being different from the other elements on the page
src src="www.example.com" The src attribute specifies the URL of the page to show in the iframe
height height="123" Specifies the height of the iframe in pixels
width width="123" Specifies the width of the iframe in pixels
scrolling scrolling="yes/no" Indicates whether the iframe should have scrollbars or not. Not supported in HTML5
frameborder frameborder="0/1" Indicates whether the iframe should have a border or not. 0 = no, 1 = yes. Not supported in HTML5
seamless seamless="seamless" In HTML5, seamless can be applied to an iframe where scrollbars are not desired
name name="description" Contains a description of the page, and is used by search engines to understand what the page is about
name name="keywords" Contains a list of comma-separated words that a user might search on to find the page
name name="robots" This indicates whether search engines should add this page to their search results or not(content="nofollow" for yes, content="noindex" for no)
http-equiv http-equiv="author" This defines the author of the web page
http-equiv http-equiv="pragma" This prevents the browser from caching the page
http-equiv http-equiv="expires" This option is used to indicate when the page should expire, and no longer be cached

Flash, Video, and Audio Tags

Tag Name Description
<video> Video This element has a number of attributes wich allow you to control video playback
<audio> Audio This element has a number of attributes wich allow you to control audio playback
<source> Source Nested within the video or audio elements, this element specifies the location of file to be played

Flash, Video, and Audio Attributes

Attributes Content Description
src src="example.mp4" This specifies the path to the video and/or audio
poster poster="example.jpg" This allows you to specify an image to show while the video is downloading, or until the user tells the video to play
width, height width="123" height="123" These specify the size of the player in pixels
controls controls When used, this indicates to the browser, that it should supply its own controls for playback
autoplay autoplay When used, this specifies that the file should play automatically
loop loop When used, this indicates that the video and/or audio should start playing again once it has ended
preload preload="none" Tells the browser not to load the video and/or audio until the user presses play
preload preload="auto" Tells the browser to download the video and/or audio when the page loads
preload preload="metadata" Tells the browser to just collect information such as the size, first frame, track list, and duration
type type="video/mp4" This attribute tells the browser what format the video and/or audio is in
codecs codecs="example" The codec that was used to encode the video is supplied within the type attribute
